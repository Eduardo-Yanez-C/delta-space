Licencia requerida

En este equipo aún no hay licencia registrada, o la anterior ya no es válida. Seleccione el archivo de licencia entregado por el administrador (primera instalación o renovación). No escriba códigos a mano: la licencia es un archivo (.json o .lic) que usted recibe y guarda en su equipo.

Formatos aceptados: .json, .lic. La licencia debe incluir en el archivo firmado el mismo ID de instalación que muestra esta pantalla y una fecha de vencimiento (validUntil). Si el archivo fue editado, la firma fallará; use el archivo tal como se lo enviaron.

Acceso desarrollador

Con el instalador que incluye la API en este mismo PC, puede activar una licencia temporal sin archivos: ingrese su correo y contraseña (cuenta ADMIN_DEV en el sistema), elija los días y pulse el botón. La validación ocurre en la API local; la contraseña no se guarda. Si usa otro emisor, el administrador define la URL con variable de entorno.

No hay API embebida ni URL de emisor configurada. Use «Seleccionar licencia» o defina DESKTOP_DEV_LICENSE_ISSUER_URL (solo instalaciones especiales).
